# Contributing to Langdock Examples

Thank you for your interest in contributing! This repository is a community collection of ideas, examples, and tools for using Langdock. Your contributions help others discover creative ways to work with AI.

## How to Contribute

### 1. Fork the Repository

Click the "Fork" button at the top right of this repository to create your own copy.

### 2. Clone Your Fork

```bash
git clone https://github.com/YOUR-USERNAME/langdock-examples.git
cd langdock-examples
```

### 3. Create a Branch

```bash
git checkout -b my-new-example
```

### 4. Add Your Contribution

Add your files in an appropriate folder structure. See guidelines below.

### 5. Commit Your Changes

```bash
git add .
git commit -m "Add: description of your contribution"
```

### 6. Push and Create a Pull Request

```bash
git push origin my-new-example
```

Then go to GitHub and click "Create Pull Request".
