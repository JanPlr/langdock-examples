# Langdock API Examples

This folder contains examples for using the Langdock API to integrate AI capabilities into your applications.

## Available Examples

| Example | Description |
|---------|-------------|
| [image-generation](./image-generation/) | Generate images using the Langdock Assistant API |

## Getting Started

Each example has its own folder with:
- A dedicated `README.md` with setup instructions
- A `requirements.txt` with Python dependencies
- Example code files

Navigate to the example folder you're interested in and follow the instructions there.

## API Documentation

For more information about the Langdock API, visit:
- [API Introduction](https://docs.langdock.com/api-endpoints/api-introduction)
- [Assistant API](https://docs.langdock.com/api-endpoints/assistant/assistant)
- [OpenAI Completion API](https://docs.langdock.com/api-endpoints/completion/openai)

## Getting an API Key

1. Go to your Langdock workspace settings
2. Click on "API" under products
3. Create a new API key
4. Copy the key and use it in the examples

See the [API Key Guide](https://docs.langdock.com/api-endpoints/assistant/assistant-api-guide) for detailed instructions.
